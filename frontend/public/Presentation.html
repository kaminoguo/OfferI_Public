How It Works - Our Workflow

Intelligent Multi-Stage Process

Stage 1

Initial Targeting

1. User inputs background/preferences

2. DB Query #1: Retrieve university shortlist

3. Web search + LLM knowledge → School selection

Stage 2

Program Discovery

1. DB Query #2: Extract ALL programs from selected schools

2. First Filter: Program name matching

Stage 3

Deep Screening

1. DB Query #3: Program description, duration, career outcomes

2. Second Filter: Detailed criteria matching

3. LLM Web Search: Third screening with latest information

Stage 4

Final Recommendation

1. LLM scores remaining programs

2. Deep Search: Top 10 programs get thorough analysis

3. Final Report: Personalized, actionable recommendations
